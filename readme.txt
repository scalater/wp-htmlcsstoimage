== Installation ==
Upload the entire Recipe Revolution folder to the /wp-content/plugins/ directory or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' menu in WordPress.
Head to the 'Recipe Revolution Settings' menu item in your admin sidebar to adjust the settings
